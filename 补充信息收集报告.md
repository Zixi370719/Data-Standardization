### 1.2 微信公众号内容手动收集指南

**建议用户手动操作步骤：**

#### 方法一：微信APP直接访问
```
1. 打开微信APP
2. 点击右上角"+" → "添加朋友"
3. 选择"查找公众号"
4. 搜索以下关键词（逐个搜索）：
   - 全国数据标准化
   - 全国数标委
   - 中国标准化
   - 信标委
   - 大数据标准工作组
5. 点击公众号进入 → 关注（如未关注）
6. 点击"消息" → 查看"历史消息"
7. 在历史消息中筛选时间：2026年1月、2月、3月
8. 重点关注文章标题包含：
   - 工作动态
   - 会议通知
   - 标准发布
   - 政策解读
   - 成员单位新增
   - DCMM评估
   - WG1/WG2/WG4/WG5工作组
   - AG3应用组
9. 对重要文章进行以下操作：
   - 点击右上角"..." → 选择"复制链接"
   - 或使用长截图功能保存
   - 或使用"收藏"功能暂存
```

#### 方法二：使用微信PC版
```
1. 在电脑上打开微信PC版
2. 在搜索框输入公众号名称
3. 进入公众号 → 查看历史消息
4. 更方便查看和复制内容
5. 可以使用浏览器插件（如微信读书/微信文章链接提取工具）
```

#### 方法三：使用第三方工具（非官方）
```
参考工具（需自行验证合法合规性）：
- 微信公众号文章导出助手
- WeChat Article Extractor
- 公众号历史消息查看器

⚠️ 注意：使用第三方工具需遵守微信用户协议和版权法
⚠️ 注意：不推荐使用破解工具或非法爬虫
```
